🌍 GroundTruth™ - By TitanicX
-----------------------------

Natural Disaster Management System - Powered by Community, Backed by Authorities.
----------------------------------------------------------------------------------

GroundTruth™ is a comprehensive, real-time natural disaster management platform designed to bridge the gap between civilians and emergency authorities. Developed by TitanicX, this application empowers users to report disaster-related incidents while enabling authorities to take swift and informed action.

🚨 Key Features:-
------------


🧑‍💼 Dual Access Dashboards
-----
GroundTruth™ offers two separate dashboards with distinct roles and privileges:
--
👤 User Dashboard:-
---

File Reports: Users can report disaster-related incidents by typing, voice-to-text, or uploading a video.

Voice Commands: Say commands like “Open Map” or “Open Live Feed” to interact hands-free.

Live Maps: View real-time incident markers based on user reports.

Community Chat: Communicate and collaborate with other users during emergencies.

Reports Section:

Bar graph showing disaster types & frequency.

Live feed of all reports with details (coordinates, status, reporter, time, etc.).

Notifications: View alerts and updates posted by admins.

Live News: Stay updated with real-time news from trusted sources like the Ministry of Government of India.

Directory & Safety Resources: Quick access to emergency contact numbers (Police, Hospital, Ambulance, Fire Brigade).

Theme Support: Toggle between light and dark mode.

Multilingual Support: Supports 150+ languages for inclusive access.

AI ChatBot (TitanicX Owned):

End-to-end encrypted conversations.

Get guidance, ask disaster-related questions, or seek help.

🛡️ Admin Dashboard
--------------

Admins have advanced capabilities to manage the platform efficiently:

Moderate User Reports: Delete inappropriate or misleading reports.

Post Notifications: Push urgent messages to users' dashboards as popup alerts.

Chatbot Controls: Start, stop, or restart the AI chatbot from the admin panel.

User Moderation: Report users to reduce their honor score by 1 credit.

Dispatch Alert System: Classify incidents as Minor, Major, or Very Serious for prioritization.

Notification Manager: Create, publish, and delete notifications as needed.

Automatically fetches nearby emergency services based on user’s live location (with permission).

Post Verification: Posts are labeled as Verified, Under Investigation, or Fake based on admin review.
