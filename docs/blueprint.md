# **App Name**: Verdant Sentinel

## Core Features:

- User Authentication: Secure user login and registration to manage access to the app.
- Disaster Update Submission: Users can submit text-based updates, and attach media (photos or videos) regarding natural disasters, specifying the disaster type and location.
- Community Updates Feed: Display real-time updates in a chronologically ordered feed, showing the most recent posts at the top.
- Geographic Tooltip Summary: LLM summarizes incoming update information to automatically identify affected locations using a geographic coordinates tool. Users can mouseover a given disaster location and get a summary of what's happening.
- Interactive Disaster Map: Display an interactive map visualizing ongoing disasters using colored pins to mark disaster locations and basic info.
- Disaster Type Filters: Users can filter updates based on disaster type (e.g., flood, earthquake, fire) to focus on specific events.

## Style Guidelines:

- Primary color: Deep teal (#388E8E) for conveying trustworthiness and a connection to nature, like the earth and sea. Not teal; it leans green, to avoid negative cliches around teal.
- Background color: Light gray (#F0F0F0), a desaturated version of the primary color. (Hue is almost identical to primary, saturation 15%, brightness increased for a light theme.)
- Accent color: Burnt orange (#D97706) to highlight important updates and calls to action (CTAs). Analogous to teal/green on the color wheel; this is achieved by shifting the hue 30 degrees.
- Headline font: 'Space Grotesk' sans-serif for headlines; body font: 'Inter' sans-serif for main content; Space Grotesk offers a techy/scientific look.
- Use outlined icons from a set like Font Awesome or Material Icons to represent disaster types and update categories.
- Employ a clean, card-based layout for updates, prioritizing readability and visual hierarchy. The disaster map should fill approx. 60% of the above-the-fold, with cards to the right of the fold and below.
- Use subtle transitions and animations for loading updates and map interactions to enhance user experience without being distracting.
